# Seckill-System

#### 介绍
秒杀系统。


#### 软件架构
软件架构说明
1.  功能：登录，商品列表，秒杀
2.  优化：拦截器，内存，分布式锁，消息队列，限流等
3.  技术栈：Java，MySQL，MyBatisPlus，Redis，RabbitMQ等

#### 使用场景

高并发，分布式

#### 解决问题

1. 库存超卖
2. 下单不一致
3. 分布式服务器
4. 减少数据库、内存访问优化
5. 页面静态化
